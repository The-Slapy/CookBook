## Uso

Abre el archivo [index.html](index.html) en tu navegador para iniciar la aplicación.

## Estructura de Archivos

- **css/styles.css**: Contiene los estilos CSS para la aplicación.
- **fallas/fallas_bd.js**: Contiene la base de datos de fallas y soluciones.
- **html/**: Contiene las diferentes páginas HTML de la aplicación.
- **images/**: Contiene las imágenes de fallas utilizadas en la aplicación.
- **js/main.js**: Contiene el código JavaScript principal para la funcionalidad de la aplicación.
- **js/libs/jszip.min.js**: Biblioteca para la manipulación de archivos ZIP.
- **pdfs/pdf_bd.js**: Contiene la base de datos de PDFs.
- **index.html**: Página principal de la aplicación.

## Funcionalidades

- **Navegación**: Menú lateral para navegar entre diferentes secciones.
- **Búsqueda**: Funcionalidad de búsqueda para encontrar problemas y soluciones específicas.
- **Visualización de PDFs**: Carga y visualización de documentos PDF.
- **Formulario**: Formulario para agregar nuevas fallas y soluciones.
